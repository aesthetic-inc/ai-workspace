## Lesson 2: 変数とデータ型

### 学習目標（観察可能）
1. 変数の概念を説明できる
2. 変数を使って値を保存・呼び出しできる
3. 文字列型と数値型の違いを区別できる
4. 変数を使った簡単なプログラムを作成できる

### 時間配分（30分）
- **導入（3分）**: 前回の復習、変数とは何か
- **本編（22分）**:
  - 変数の基本（7分）
  - データ型（文字列・数値）（8分）
  - 実践プログラム作成（7分）
- **まとめ（5分）**: ミニプロジェクト発表、振り返り

### コンテンツ

#### 1. 変数の基本
```python
# 変数は「箱」のようなもの
name = "太郎"
age = 14
print(name)
print(age)

# 変数を使った計算
height = 160
next_year_height = height + 5
print(next_year_height)
```

**概念説明**:
- 変数名は意味のある名前をつける
- `=` は「代入」を意味する
- 変数は何度でも更新できる

**練習問題**:
1. 自分の情報を変数に保存（名前、年齢、好きな科目）
2. 変数を使って自己紹介文を表示

#### 2. データ型
```python
# 文字列型（str）
message = "こんにちは"
print(message)

# 数値型（int, float）
score = 85        # 整数
pi = 3.14         # 小数
print(score)
print(pi)

# 型の確認
print(type(message))  # <class 'str'>
print(type(score))    # <class 'int'>
```

**注意点**:
```python
# これはエラーになる
result = "10" + 5  # 文字列と数値は足せない

# 正しい方法
result = int("10") + 5  # 15
```

**練習問題**:
1. 文字列と数値を使った変数を3つずつ作成
2. `type()`を使って確認

#### 3. 実践：簡単な計算プログラム
```python
# 買い物の合計計算プログラム
apple_price = 120
orange_price = 150
quantity = 3

total = (apple_price + orange_price) * quantity
print("合計金額:", total, "円")

# または
print(f"合計金額: {total}円")  # f-string（やや高度）
```

**ミニプロジェクト**:
「お小遣い管理プログラム」
- お小遣い額を変数に保存
- 使った金額を変数に保存
- 残りを計算して表示

### アクティビティ
1. **変数ゲーム**: 変数名当てクイズ（良い名前・悪い名前）
2. **タイピング練習**: コードを見ながら正確に入力
3. **ミニプロジェクト**: お小遣い管理プログラムを作成
4. **発表タイム**: 作ったプログラムを隣の人に見せる

### 教材

## Lesson 3: 条件分岐と入力処理

### 学習目標（観察可能）
1. `input()`を使ってユーザーからの入力を受け取れる
2. `if`文の構文を正しく書ける
3. 条件に応じて異なる処理を実行するプログラムを作成できる
4. 簡単な対話型プログラムを完成させられる

### 時間配分（35分）
- **導入（3分）**: 前回の復習、今日のゴール紹介
- **本編（27分）**:
  - ユーザー入力（5分）
  - if文の基本（8分）
  - if-else, if-elif-else（9分）
  - 総合プロジェクト（5分）
- **まとめ（5分）**: 成果発表、3回の振り返り

### コンテンツ

#### 1. ユーザー入力
```python
# input()で入力を受け取る
name = input("名前を入力してください: ")
print("こんにちは、" + name + "さん!")

# 数値として受け取る
age = input("年齢を入力してください: ")
age = int(age)  # 文字列を数値に変換
print("あなたは", age, "歳です")
```

**練習問題**:
1. 好きな食べ物を入力させて表示
2. 2つの数値を入力させて合計を表示

#### 2. if文の基本
```python
# 条件分岐
age = 14
if age >= 13:
    print("あなたは中学生です")

# 比較演算子
# == 等しい
# != 等しくない
# > より大きい
# < より小さい
# >= 以上
# <= 以下

score = 85
if score >= 80:
    print("よくできました！")
```

**練習問題**:
1. 点数を入力させて、80点以上なら「合格」と表示
2. 年齢を入力させて、13歳以上なら「中学生以上」と表示

#### 3. if-else と if-elif-else
```python
# if-else
score = int(input("点数を入力: "))
if score >= 60:
    print("合格です！")
else:
    print("不合格です...")

# if-elif-else（複数条件）
score = int(input("点数を入力: "))
if score >= 90:
    print("優秀！")
elif score >= 70:
    print("良好")
elif score >= 60:
    print("合格")
else:
    print("不合格")
```

**練習問題**:
1. 温度を入力させて、暑い/ちょうどいい/寒いを判定
2. 好きな季節を入力させて、それぞれの季節のコメントを表示

#### 4. 総合プロジェクト：診断プログラム
```python
# 「今日のラッキー度診断」
print("=== 今日のラッキー度診断 ===")
name = input("名前を入力: ")
number = int(input("1-10の好きな数字を入力: "))

if number >= 8:
    print(f"{name}さん、今日は超ラッキー！")
elif number >= 5:
    print(f"{name}さん、今日はまあまあラッキー")
else:
    print(f"{name}さん、今日は普通の日です")

print("診断終了！")
```

**発展プロジェクト例**:
- 性格診断プログラム
- おすすめ映画診断
- 簡単なクイズゲーム

### アクティビティ
1. **ライブコーディング**: 講師が段階的に診断プログラムを作成
2. **ペアプログラミング**: 2人1組で協力してプログラム作成
3. **オリジナル診断作成**: 自分だけの診断プログラムを考案
